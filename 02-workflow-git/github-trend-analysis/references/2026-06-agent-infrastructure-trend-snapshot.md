## Trend Observations

1. **Token cost drives innovation** — Headroom, Codegraph, Markitdown all target reducing LLM input costs
2. **Security becoming mandatory** — NVIDIA's SkillSpector signals enterprise seriousness
3. **Knowledge graphs replacing simple RAG** — Pre-built, incrementally-updatable, interactive graphs
4. **Skill standardization emerging** — From "prompt engineering" to reusable "skill files"
5. **Strong Chinese developer participation** — Agent-Reach (Xiaohongshu/Bilibili), MoneyPrinterTurbo, TencentDB Memory
6. **Local-first trend** — Multiple projects emphasizing "100% local", "zero external API"
