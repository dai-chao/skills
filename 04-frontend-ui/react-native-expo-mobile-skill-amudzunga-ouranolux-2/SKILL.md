# React Native Expo Mobile Skill

## Stack

```text
React Native
Expo Development Builds
TypeScript
Expo Router
Zustand
TanStack Query
Axios
Zod
React Hook Form
MMKV
SQLite
SecureStore
Reanimated
Gesture Handler
Skia
Sentry
Firebase/FCM or Expo Notifications
```

## Rules

- TypeScript everywhere.
- API response validation with Zod.
- Server state belongs in TanStack Query.
- UI/local preferences belong in Zustand/MMKV.
- Auth secrets belong in SecureStore.
- No mobile-to-ML direct calls.
- No gambling/wagering functionality.

Read `folder-structure.md`.
