# AI 视频推广 Prompt 模板

> 适用于即梦、可灵、Runway 等支持 `@图片1` / `@视频1` / `@音频1` 多模态参考的 AI 视频工具。

## 通用结构

给用户的输出应包含 5 部分：

1. **素材映射表**（最多 12 个）
2. **主输入 Prompt**（可直接复制到 AI 工具）
3. **分镜脚本**（按时间线）
4. **风格关键词**
5. **纯文本 Fallback Prompt**（无参考素材时使用）

---

## 示例：科技产品 15 秒宣传视频

### 素材映射

| 素材 | 类型 | 内容 |
|------|------|------|
| @图片1 | 图片 | 产品 logo 或官网主视觉 |
| @图片2 | 图片 | 深色科技感背景（数据流、粒子、网格） |
| @图片3 | 图片 | 设备桌面 + 相关软件图标 |
| @视频1 | 视频 | 镜头推进 / 扫描线扫过的科技素材 |
| @音频1 | 音频 | 沉稳男声旁白样本 |
| @音频2 | 音频 | 电子/科技感背景音乐 |

### 主输入 Prompt

```
@图片1 以发光粒子效果从 @图片2 的深色数据流背景中浮现，@视频1 的镜头推进感扫过屏幕，切入一台 MacBook 桌面，@图片3 的 AI 客户端图标依次亮起。屏幕边缘闪现红色警告波纹，随后被蓝色扫描光圈覆盖，风险标记转为绿色对勾。最后定格在 Agent Guard 产品界面，出现“免费扫描”和“一键修复 Pro”按钮。旁白音色参考 @音频1，背景音乐参考 @音频2。
```

### 分镜脚本

| 时间 | 画面 | 旁白/文案 |
|------|------|------------|
| 0-3s | 粒子聚合出 logo，背景数据流流动 | “AI 工具越用越顺手，但账号可能正在被风控。” |
| 3-7s | 桌面 AI 图标亮起，红色警告闪烁 | “网络、指纹、凭证、客户端身份——任何异常都可能被封。” |
| 7-11s | 蓝色扫描光圈覆盖，红色转绿色 | “Agent Guard 一键扫描本机环境，定位风险。” |
| 11-15s | 产品界面定格，CTA 按钮高亮 | “免费扫描，降低封号风险。” |

### 风格关键词

科技、安全、冷静、高级感，深蓝色 + 黑色主色调，UI 清晰，运镜平稳推进，避免花哨转场。

### 纯文本 Fallback Prompt

```
15秒科技风格产品宣传视频：深色数据流背景，Agent Guard logo 以发光粒子聚合出现，镜头推进到 MacBook 桌面，Claude/Cursor 等 AI 工具图标依次亮起，屏幕边缘闪现红色警告波纹，随后蓝色扫描光圈覆盖整个界面，风险提示转为绿色对勾，最后定格产品界面并显示“免费扫描”和“一键修复 Pro”按钮。旁白沉稳男声，电子科技感背景音乐。
```

---

## 输出约定

- 主 prompt 中 `@图片N` / `@视频N` / `@音频N` 必须与素材映射表中的编号一致。
- 分镜脚本的时长总和应与用户要求的视频时长一致（常见 15s / 30s / 60s）。
- 如果用户未提供参考素材，必须同时提供纯文本 fallback prompt。
- 旁白文案和画面描述应一一对应，避免音画不同步。
